# 🕒 Aesthetic Digital Clock

A minimal and modern digital clock built using **HTML, CSS, and JavaScript**. The application features a clean glassmorphism-inspired interface with real-time updates, responsive design, theme switching, and 12/24-hour time format support.

🌐 **Live Demo:** https://aaliyahalics23021.github.io/Aesthetic-Digital-Clock/

---

## ✨ Features

- 🕒 Real-time digital clock
- 🌙 Light/Dark theme toggle
- ⏰ 12-hour & 24-hour format switch
- 📱 Responsive design
- 🎨 Modern glassmorphism UI
- ⚡ Lightweight and fast
